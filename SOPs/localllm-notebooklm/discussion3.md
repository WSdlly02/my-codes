  我不建议把“专题正文内容”放在正文 atom 后面，因为专题索引是索引，不是正文。它应该放在教程正文之后、补充正文之前，作为横向查找入口。

  更准确地说：

  总索引 -> 章节索引 -> 主干正文 -> 专题索引 -> 补充正文

  为什么这样排

  考试时最常见路径是：

  先查总索引
  -> 找到章节索引或专题索引
  -> 再翻 atom 正文

  所以总索引必须最前。章节索引紧随其后，因为它是课程主干。主干正文放在专题索引前，是因为课程考试大概率以章节逻辑为基础。专题索引放在主干正文后，可以横向补充。补
  充资料正文最后放，避免打乱主干。

  PDF 转换工具

  优先建议用 Pandoc + XeLaTeX：

  pandoc final_print.md \
    -o final_print.pdf \
    --pdf-engine=xelatex \
    -V CJKmainfont="Noto Serif CJK SC" \
    -V geometry:margin=1.6cm \
    --toc \
    --number-sections

  页码由 LaTeX 自动加。中文效果比浏览器打印稳定。

  如果你想快速实现，也可以用 HTML/CSS + Playwright/Chrome 打印，但页码、目录、长表格分页不如 LaTeX 稳。
